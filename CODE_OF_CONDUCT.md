# Code of Conduct

## Ethical Guidelines

### 1. Legal Compliance
- Only use these tools on systems you own or have explicit written permission to test
- Never attack production systems without authorization
- Respect all applicable laws and regulations

### 2. Responsible Disclosure
- Report vulnerabilities to vendors through proper channels
- Allow reasonable time for fixes before public disclosure
- Never exploit vulnerabilities for personal gain

### 3. Educational Use Only
- These tools are for learning and research
- Use in controlled environments (labs, CTFs, authorized systems)
- Document your learning process

### 4. Privacy Respect
- Never access, modify, or delete data without permission
- Respect user privacy and confidentiality
- Anonymize any data in research publications

## Violations
Violations of this code will result in:
1. Immediate removal of contributions
2. Reporting to authorities if illegal activity is suspected
3. Permanent ban from the repository
