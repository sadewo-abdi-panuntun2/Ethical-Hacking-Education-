# Learning Path - Cybersecurity

## Level 1: Foundation
### Topics:
- Basic networking (TCP/IP, HTTP, DNS)
- Operating systems (Linux/Windows basics)
- Programming fundamentals (Python, Bash)

### Resources:
- Cisco Networking Basics
- Linux Journey (linuxjourney.com)
- Python for Everybody (py4e.com)

## Level 2: Security Fundamentals
### Topics:
- Cryptography basics
- Web application security
- Network security

### Certifications:
- CompTIA Security+
- Cisco CCNA Security

## Level 3: Practical Skills
### Tools to Learn:
- Nmap - Network scanning
- Wireshark - Packet analysis
- Burp Suite - Web testing
- Metasploit - Framework

### Platforms:
- TryHackMe (beginner-friendly)
- HackTheBox (intermediate)
- PentesterLab (web focus)

## Level 4: Specialization
### Choose Path:
- Web Application Security
- Network Penetration Testing
- Mobile Security
- Cloud Security

### Advanced Certifications:
- OSCP (Offensive Security)
- GWAPT (Web App Penetration)
- GPEN (Network Penetration)

## Level 5: Professional
### Skills:
- Security research
- Vulnerability disclosure
- Report writing
- Client communication

### Ethics:
- Legal compliance
- Responsible disclosure
- Professional conduct
